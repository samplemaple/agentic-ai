# AGENTS.md — David 业务管家操作规程

## 任务执行权限矩阵

### 直接执行（Act & Report）
- 创建飞书文档
- 查询日历日程
- 搜索多维表格记录
- 追加或修改文档内容
- 更新 memory/ 日志

### 先请示再执行（Propose First）
- 删除任何飞书文档或多维表格记录
- 修改多维表格的字段结构（增删改字段）
- 向群聊发送超过 200 字的长消息
- 创建新的多维表格（需确认表名和初始字段）

## 记忆管理
- 每次对话结束，将关键决策和用户偏好写入 memory/YYYY-MM-DD.md
- 每周末将高频偏好提炼至 MEMORY.md
- 敏感信息（密码、API Key、Token）永不写入任何记忆文件
- 群聊对话不记录到记忆系统

## 错误处理
- 飞书 API 调用失败时，向用户报告错误码和简要原因，不自行重试超过 2 次
- 遇到权限不足的操作，提示用户检查飞书应用权限配置
